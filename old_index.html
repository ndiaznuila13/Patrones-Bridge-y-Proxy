error_outline

The Problem

Imagine a messaging system. You have different platforms (Web, Mobile) and different message types (Text, Image). If you use inheritance, you get an explosion of classes: WebTextMsg, WebImageMsg, MobileTextMsg, MobileImageMsg... Adding a new platform or type multiplies the complexity.

lightbulb

The Solution

Bridge separates the abstraction (Platform) from the implementation (Message Type). The abstraction holds a reference to the implementation, delegating work to it.

code TypeScript Implementation 


interface MessageSender {
    sendMessage(text: string): void;
}

class EmailSender implements MessageSender {
    sendMessage(text: string) { console.log(`Email: ${text}`); }
}

class SlackSender implements MessageSender {
    sendMessage(text: string) { console.log(`Slack: ${text}`); }
}

abstract class NotificationPlatform {
    protected sender: MessageSender;
    constructor(sender: MessageSender) { this.sender = sender; }
    abstract notify(message: string): void;
}

class UrgentNotification extends NotificationPlatform {
    notify(message: string) {
        this.sender.sendMessage(`URGENT: ${message}`);
    }
}

// Usage
const emailSender = new EmailSender();
const urgentEmail = new UrgentNotification(emailSender);
urgentEmail.notify("Server is down!");
error_outline

The Problem

You have a massive object that consumes significant system resources (e.g., a huge database query or a heavy 3D model). You need it occasionally, but instantiating it upfront degrades application startup time or memory footprint.

lightbulb

The Solution

Create a substitute (Proxy) that implements the same interface as the real object. The Proxy controls access, instantiating the real object only when strictly necessary (Lazy Loading).

code TypeScript Implementation 


interface DatabaseQuery {
    execute(): string;
}

class HeavyDatabaseQuery implements DatabaseQuery {
    constructor() {
        // Expensive connection setup
        console.log("Connecting to DB...");
    }
    execute(): string {
        return "Heavy data payload";
    }
}

class DatabaseProxy implements DatabaseQuery {
    private realQuery: HeavyDatabaseQuery | null = null;
    
    execute(): string {
        if (this.realQuery === null) {
            this.realQuery = new HeavyDatabaseQuery();
        }
        console.log("Proxy: Logging query execution.");
        return this.realQuery.execute();
    }
}

// Usage
const proxy = new DatabaseProxy();
// Connection happens ONLY here, not on instantiation
console.log(proxy.execute());
